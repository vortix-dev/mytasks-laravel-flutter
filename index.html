About MyTasks

MyTasks is a simple yet powerful task management application that allows users to create, update, and track their tasks easily. Whether you're using the mobile app or the web platform, you'll get a smooth and fast experience. With features like due dates, priorities, and filtering by task status, MyTasks helps you stay in control of your productivity.

The app leverages the latest technologies to ensure performance and security, including Laravel 11 for backend, ReactJS for web, and Flutter for mobile with MySQL as the database engine.
